Dependencies
============

   * Python 3 (tested with v3.6.5 - https://www.python.org/downloads/)
   * Pip version 3 (package manager)
   * PyAbel
   * NumPy
   * Scipy
   * Matplotlib
   * Tom Schoonjan's xraylibs (https://github.com/tschoonj/xraylib).
